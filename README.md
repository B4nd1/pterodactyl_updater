# pterodactyl_updater
Script to easily update Pterodactyl Panel &amp; Wings

It follows the official update steps from the documentation as of Feb 2026.

[Panel manual upgrade](https://pterodactyl.io/panel/1.0/updating.html)

[Wings manual upgrade](https://pterodactyl.io/wings/1.0/upgrading.html)


Use this script at your own risk. It is provided as-is without warranty. Always keep backups of your database and files before running it.
